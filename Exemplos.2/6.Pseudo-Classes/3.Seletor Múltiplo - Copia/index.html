Navegadores (Browsers)

Para navegar pela Web é necessário um software cliente que chamamos de navegador (em inglês, Browsers). Ele é responsável por: solicitar documentos na Internet e interpreta-los, exibindo-os para o usuário.

Exemplos de navegadores:

  • Netscape
  • Internet Explorer
  • Mosaic
  • Lynx
  • Opera
